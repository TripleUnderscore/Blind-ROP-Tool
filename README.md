# Blind-ROP-Tool

Outil permettant de faire des Blinds ROPs. Pour le moment il ne permet que de cibler des binaires x64 de type ELF, l'idée étant à termes de rendre l'outil capable de performer sur plusieurs architectures.

Pour le moment pas de man, un peu la flemme, mais ```./brop.py -h``` renvoie l'aide nécessaire.

Le code est simpliste, je l'écris surtout pour apprendre à coder, et plusieurs commentaires figurent dedans dans le but de l'optimiser.
